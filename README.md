## Sobre o projeto

- Projeto do curso da EBAC com finalidade no aprendizado de Micro Frontend.

## Como utilizar

- Criar um terminal para cada pasta micro frontend (Cart, Container e Menu) e instalar as dependências:
  -  Abrir terminal
  -  cd cart (cd menu / cd container)
  -  npm install
  -  npm run dev
  -  Repetir o processo até ter 3 terminais abertos.

-  Após abrir os três terminais, acessar o localhost do cd container.

## Como funciona

- Cada micro frontend se comunica através de eventos globais do navegador, eliminando a necessidade de props e permitindo que cada aplicação funcione de forma independente, mas integrada ao sistema final.
